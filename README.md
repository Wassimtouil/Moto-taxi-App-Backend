## 🚀 Overview

Moto-Taxi Backend is a **Spring Boot REST API** that powers a smart motorcycle taxi platform. It provides secure authentication, ride management, real-time communication, geolocation services, wallet and payment management, email notifications, and AI-powered assistance for administrators.

---

## ✨ Key Features

* 🔐 JWT Authentication & Authorization
* 👤 Customer, Driver & Admin Management
* 🛵 Ride Booking & Trip Management
* 📍 Real-Time Driver Tracking
* 🌍 Nearby Driver Search
* 🗺️ Route & Distance Calculation (Mapbox)
* 💳 Wallet & Payment Management
* 📧 Email Notifications & OTP Verification
* 🤖 AI Chatbot Integration
* 📡 RESTful API
* 🛡️ Secure & Scalable Architecture
